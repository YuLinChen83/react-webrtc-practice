# WebRTC Practice
[Demo](https://react-webrtc-practice.vercel.app/)  / [Note]()  

1-to-1 video chat webpage built with WebRTC, React and Firestore.

## Available Commands
* `npm install`
* `npm run dev`

## Built With
* React 7.0.0
* Typescript 4.3.2
* Firebase 9.0.2
* Tailwindcss 2.2.15
* Vite 2.5.4
